# CTF VM

An Ubuntu 14.04 x64 VM for binary exploitation CTF problems.

## Installation

```
git clone git@github.com:gsingh93/ctf-vm.git
cd ctf-vm
vagrant up
```

## Packages

Included packages

- pwndbg
- peda
- binjitsu (pwntools)
- Pin
- AFL
- ROPgadget
- rp++
- xrop
- QEMU with ARM support
- angr
