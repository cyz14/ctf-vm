# CTF VM

An Ubuntu 14.04 x64 VM for binary exploitation and reversing CTF problems. For a 16.04 VM, checkout the `ubuntu16.04` branch.

## Installation

```
git clone git@github.com:gsingh93/ctf-vm.git
cd ctf-vm
vagrant up
```

## Packages

Included packages

- pwndbg
- peda
- pwntools
- Pin
- AFL
- ROPgadget
- rp++
- xrop
- QEMU with ARM support
- angr
